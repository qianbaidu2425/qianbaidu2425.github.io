#该程序用于验证伯努利大数定律
import random
import matplotlib.pyplot as plt
#假设在每次独立实验中，事件发生的概率为0.5
p = input("Please type in the probability of the event'A': ")
p = float(p)
#创建空列表用来存放独立重复实验
experiment = []

#创建好散点图
plt.scatter([],[])
plt.title('Frequency-Exp_times')
plt.xlabel('Exp_times')
plt.ylabel('Frequency')

#假设实验的重复次数为n,模拟n次独立重复实验
for n in range (1,10001):
    print(n)
    experiment = []
    for i in range (1,n+1):
        num = random.uniform(0,1)
        if num <= p:
            value = 1
        if num > p:
            value = 0
        experiment.append(value)
    nA = sum(experiment)
    plt.scatter(n,nA/n,color='black',s=2)
    
#画出散点图
plt.show()

读《桶川跟踪狂杀人事件有感》
2023/11/24

这几天正好道法课大作业的ddl将近，同学推荐了一本书《桶川跟踪狂杀人事件》，就着这本书把大作业写掉了。

背景提要

这是有关一个女孩被人杀害了三次的故事，第一次是凶手，第二次是媒体，第三次是警方。

1999年10月26日，年轻女子猪野诗织在日本埼玉县JR桶川站前遭人刺杀。任职于新潮社《FOCUS》周刊的记者清水洁像往常一样展开了对该案的调查。然而，随着调查的深入，清水洁逐渐发现了这起案件背后种种骇人听闻的真相和令人发指的黑幕。经过长达四年的不懈努力，这起案件才逐渐尘埃落定。清水先生将他追踪此案的全过程记录在《桶川跟踪狂杀人案》一书中。

世人从来向往“天下大同”的理想社会。桶川一案揭露出的残酷现实，表明建设和谐社会仍道阻且长，需要但笔者认为清水先生的这本著作能为迷茫中的我们指引一些正确的方向。

社会秩序的维持离不开法治

猪野诗织第一次是被凶手杀害的。

诗织第一次遇见小松和人时，小松和人自称他是“从事汽车销售的青年实业家”，但是小松和人实际上经营着非法色情行业，在清水记者不懈的追踪下，他发现小松和人经营的色情按摩店规模相当大。

难以想象，在即将迎来二十一新世纪的时候，这些非法产业却在日本的社会上横行霸道。事实上，早在1900年，日本就出台了《淫行禁止法》，并在1951年出台了《淫秽物贩卖等罪》这一部法律。那么，这或许说明，当时的日本池袋地区的治安部门，或许根本就没有将禁止色情产业的法律坚决实行。：

同时，在当时的日本，并没有相关法律对跟踪骚扰行为做出规定。因此，就算是遇到骚扰以至于生命受到威胁的公民，也无法指望警署对骚扰行为采取措施，书中原话如下[1]：

“跟踪狂的问题，或许警方也难以判断。实际上，警方收到这类咨询求助的数量相当多。如果被害的一方有过错，反过来遭到警方斥责，或许是活该；但诗织的案例又如何？”

一个社会缺失了法律，就缺失了评判行为的准绳。正是因为没有相关法律的规定，小松和人才可以肆无忌惮地对诗织展开跟踪和骚扰，正是因为没有相关法律的规定，才使得诗织在向警方求助时，得到的是一句句冰冷的拒绝。2000年5月18日，日本通过了《跟踪骚扰行为规范法》，那些饱受跟踪和骚扰折磨的日本公民终于拥有了捍卫自己权利的途径。

一个好的社会是有序的，而秩序的维持离不开法律的约束和执法者的尽责。在未来，我们所处的社会还会出现更多游走于“灰色地带”的行为，如果不及时制定相关的法律并坚决贯彻法治的理念，那么这些行为就有可能酿成大错。

我们需要怎样的媒体环境？

猪野诗织第二次是被媒体杀害的。

在本书中，清水先生展现了当时日本媒体传播界的扭曲。在当时的日本，民众对《FOCUS》这样的周刊的印象往往是“三流”，而某些报纸却能被冠以“一流杂志”之名。如此巨大的待遇差距，不是因为它们报道的内容存在深度上的优劣之分，而是因为——周刊不是政府宣传型的“公共媒体”。

这样的成见带来的危害，在桶川一案中被放大了。办案的搜查员提起诗织一案时，常常这么说——“那是酒家女的三流案子啦。“警方用这样不合实际的话语对媒体进行了诱导，而那些所谓的“一流杂志”只会把政府机关公布的信息照本宣科地报道出来，并把这样的形容词放大后，插在标题里，放在版面上。如此一来，诗织的形象变成了“酒店小姐”、“迷恋名牌的人”活在了许多大众心中，但她只是一个普通的女生啊……

在如今的中国社会，我们也应该避免这种情况的出现。媒体的高低应该根据其发表的内容的质量来评判，国家应当在全社会进行这种价值导向的引领。同时，新闻工作者不应该随波逐流，应该要对信息来源采取客观的态度进行审视，敢于质疑，不能因为是“官方的”就不加思考地大肆报道。

同时，笔者认为，如今的媒体不应做“标题党“。显眼的标题会让读者先入为主，错误的信息可能会存在于读者的潜意识里，潜移默化地改变大众舆论的风向。同时，笔者也认为，国家应当进一步完善审查体制，筛查出那些准确而客观的报道，不让错误的信息流入社会。

国家机关工作人员的不作为是可怕的

猪野诗织第三次诗被警方杀害的。

在本书中，最令人感到愤慨的，或许就是埼玉县上尾警署的黑暗。他们怠政，只是因为“不愿意引起麻烦”就对诗织的求助报以冷漠的态度，随意篡改笔录；他们拥有数百人组成的专案组，侦察效率却不如周刊的几位记者。在办案的过程中，他们始终绕开最重要的环节——小松和人，直到最后他们都在逼迫小松和人的哥哥小松武史承认自己是主谋，而小松和人直到最后都仅仅止于被以名誉损毁的罪嫌通缉。最可恨的是——他们把死者发生的不幸当作儿戏。他们随意扣押死者的遗物，在死者父母接到死讯之后还蛮横地要求他们配合填写各类文件，甚至还将死者生前写下的日记里的某些片段拿来咬文嚼字，拿来反击死者父母的质疑。在办案的发布会现场，有警察甚至还带着笑容，实在令人作呕。

上尾警署的警员们，不但没有拯救诗织，放任她的生命被凶手夺取，还在诗织去世后不断践踏着她的人格。
书中描写的上尉警署，是日本当时无数不作为的警署中的一个缩影，或许也是现在中国社会中许多不作为的地方政府机关的一个缩影。这些不作为的官员，在他们的恶行被公之于众之前，他们仍然在日复一日地懒政、怠政；难以想象多少人民的权益因此而收到损害。

笔者认为，我们必须进一步加大反腐的力度。国家监察机关必须深入基层，推动反腐工作的展开。同时，要给民众提供更多的反馈渠道，让更多不作为、不担责的行为被曝光出来。在全社会营造廉洁而公正的行政氛围。

总结

笔者认为，我们应该感谢清水洁先生，是他让这起拥有里程碑式意义的案件的真相能够被世人所看见。或许，我们应该感谢所有怀揣良知、正义与勇气，为了真相而奋不顾身的人们。作为后人，我们能从这本书中获得灵感，让清水先生所做出的努力化为我们建设和谐社会的良策。

16bit的感动 ep08

这集还是挺有意思的，双重穿越的剧情，而且这一集全篇都贯穿着“想象力”一词，可算是把galgame最核心的东西给表达出来了，而且这个演出一股浓浓的shaft味。期待一下这部原创还能整出什么活。

咒术回战 s2 ep18

唉，七海还是离开了，其实这个角色我一直是挺喜欢的，因为娜娜明这种成熟的气质很吸引我，而且津叔的声音我也非常喜欢，所以七海被真人杀掉的那个场面我看的是很揪心的。

说说这一集的优点吧：1、七海的蒙太奇部分做得挺好的。2、后半段打斗场面很好地把真人的癫狂的状态体现出来了。

缺点：后半段打戏的逻辑还是一如既往的糟糕，总感觉分镜有缺失；而且有一点我不能理解，就是真人遇到野蔷薇的时间点是23：16，但是后半段和虎杖的打斗的时间点是23：19，这时间点也离的太近了吧。

总而言之，我一直感觉咒术做得最好的部分还是双玉篇，五条悟和夏油杰的主线剧情和打斗的各种衔接处理都做得很好；而涉谷事变篇整体就是突出一个“乱”，感觉就是不能带着逻辑去看，不然会觉得很支离破碎，主打的就是一个意识流。

葬送的芙莉莲 ep12

这一集讲的是休塔尔克的自我成长，“汉堡排”、休塔尔克和他哥哥的故事让我挺感动的，我感觉，芙莉莲真的是把“回忆杀”这一个手法运用的炉火纯青，我从来不会觉得腻，就是：用不同时空下，同一事物在不同语境下的相似性来串联过去与现在，达到共情和感动。

很喜欢艾森在回忆中说过的一句话——“努力过的人就是战士”。

肥伦真的太可爱了，感觉她和休塔尔克真的很好磕。

不死不幸 ep08

这几集面对spoil的打戏感觉节奏确实挺奇怪的，分镜的衔接不够自然，而且就是感觉情绪渲染的不够到位，不会八濑佑树导演又开始玩之前的那套了吧？

不死不幸让我追下去的动力就是它的设定真的很有趣，每集都有新的设定，比如说这一集金箍棒都出来了，原作还是挺有东西的。

希望能不能把打斗再放的节奏快一点啊，每次都断章真的挺难受的，反正我觉得这几集做的不如前面对战“不变”的那几集，真的不想看到这么有意思的原作被David production给毁掉……

还有，花江夏树能不能少说点塑料中文啊，真的听的挺尴尬的……

间谍过家家 s2 ep08

这一集质量真是没得说，打斗+烟花的画面有机结合的非常好。而且后半部分约尔太太的走马灯的部分也做得非常好，我觉得在一部喜剧里，这一段的情感流动力是非常强的。真的太帅啦。

个人觉得间谍第二季的前几集做得比较一般，到邮轮篇这里整个厨力就上来了。

香格里拉边境 ep09

这一集路人女主终于有戏份了，不容易啊。这集开头居然没有放旁白来水时长（建议不死不幸好好学一学）。后半部分就是一般的流水式2攻略剧情，但是这个番断章是真的让人难受。每次都是遇到boss的时候断章，就真的让人挺急的……
